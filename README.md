## Features
- **Interactive 3D Background:** A custom-coded 3D wave animation using Three.js and GLSL shaders.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views using CSS Flexbox and Grid.
- **Modular Architecture:** Clean, component-based structure with separated CSS modules for scalability.
- **Photography Gallery:** A dedicated space for visual storytelling and creative work.
- **Single Page Navigation:** Smooth scrolling via `react-router-hash-link`.

## Tech Stack
- **Frontend:** React.js, Vite
- **Styling:** Custom CSS3 (Modularized)
- **Animation:** Three.js / Canvas API
- **Routing:** React Router v6
- **Deployment:** GitHub Pages
